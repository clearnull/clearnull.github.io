# 一、时间复杂度

| 时间复杂度 | 介绍                                          | 举例     |
| ---------- | --------------------------------------------- | -------- |
| O(1)       | 最底复杂度，常量值也就是                      | 哈希算法 |
| O(logn)    | 当数据增大n倍时耗时增大logn倍（log是以2为底） | 二分查找 |
| O(n)       | 数据量增大几倍，耗时也增大几倍                | 遍历算法 |
| O(n2)      |                                               |          |
